# CAFETERIA

## Descripción

Este proyecto es una página web para una cafetería que permite a los usuarios adquirir cafés en línea. El proyecto cuenta con diferentes secciones, como la página de inicio, la página de nosotros, la página del proceso, la página del menú, la página de galería y la página de contacto.

## Requisitos

Ejecutar el comando en la raiz del proyecto para la instalacion de dependencias

```
npm i
```

## Instrucciones de uso

Para acceder a la página, simplemente abra el archivo index.html en su navegador web y corrar el comando

```
gulp dev
```

para visualizar cambios agregados.
